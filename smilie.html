HTML Anchor Link and SmILiE Link Demonstration

The <a> anchor tag is the heartbeat of the web it puts the "hyper" in hypertext. Here is how to use it — and how to break it. The "flaw" in a raw HTML anchor link is that the link visited must exist to access.

The HTML Raw Anchor Tag <a>

The <a> tag is the heartbeat of the web. It defines a hyperlink, allowing users to navigate between pages or jump to specific sections of a document.

The Syntax 

<a href="https://wigil.me">Click Link to Visit William Fletcher Gilreath Home Site</a>

Key Attributes of HTML Anchor tag

  • href: Specifies the destination URL.
  • target: Specifies where to open the link (e.g., _blank).
  • rel: Defines the relationship between the current and linked document.

✅ The Successful Basic HTML Anchor Tag

A successful anchor provides clear context and uses an absolute path. The user knows exactly where they are going.

<a href="https://wikipedia.org">Visit Wikipedia</a>

Success to goto Wikipedia; a live link.

❌ The Failed HTML Anchor Tag

This anchor is a "broken promise." It links to a dead, invalid destination, leading to a 404 or dead-end.

<a href="https://boisejug.org">I am a dead link.</a>

Failure to goto boisejug.org a dead link.


The website existed in 2002 from Internet Archive.



The HTML SMart Inquire LInk Exists (SMILIE) Anchor Tag

The <a> SMILIE tag is the smart link of the web. It defines a hyperlink, allowing users to access pages even if the link is invalid (inactive, malformed, non-existent) or let the user know the link is invalid. The link color is changed to green green <a> if valid, or grey grey <a> if invalid.

The Syntax 

 <id="0" href="javascript:void(0)" data-wait=5000 data-url_head="https://www.google.xxx"
        data-url_tail="https://www.google.com" onclick="xref(id)">Invalid malformed Google link with valid alternate link</a>

Key Attributes of SmILiE link tag

  • id: The unique identifier or id (the simplest is a unique integer number) for the anchor tag.
  • data-url_head: Specifies the required primary destination URL.
  • data-url_tail: Specifies the optional secondary destination URL. (optional)
  • data-wait: Defines the optional wait time to inquire and validate a link. (optional)

👍 Smart Inquire Link Exists (SmILiE)...

The SmILiE links have three cases of failure and two cases of success to demonstrate the "smartness" of the link. For all five cases of failure and success, there is no error nor visiting non-existent, malformed, invald link.


• A SmILiE link to non-existent invalid link "boisejug.org" but with no alternate link to visit.


< id="1" href="javascript:void(0)" data-wait=5000 data-url_head="https://www.boisejug.org" onclick="xref(id)">...</a>

Invalid non-existent Link "boise.jug" No Alternate Link

• A SmILiE link to malformed URL "google.xxx" with a valid alternate link "google.de" to visit.


< id="2" href="javascript:void(0)" data-wait=5000 data-url_head="https://www.google.xxx" data-url_tail="https://www.google.de" onclick="xref(id)" >...</a>

Invalid malformed link "google.xxx" malformed link but has alternate link

• A SmILiE link to a non-existing link "boisejug.org" but alternate link visits copy on Internet Archive.


< id="3" href="javascript:void(0)" data-wait=5000 data-url_head="https://www.boisejug.org" data-url_tail="https://web.archive.org/web/20020926204018/http://www.boisejug.org/" onclick="xref(id)" >...</a>

Invalid non-existent invalid "boisejug.org" link but has alternate valid link.

• A SmILiE link to valid link "wilgil.me" to visit, but without validate alternate link not visited.


<id="4" href="javascript:void(0)" data-wait=5000 data-url_head="https://www.wilgil.me" onclick="xref(id)">...</a>

Valid link "wilgil.me" to visit but without an alternate link to visit.

• A SmILiE link to valid link "wilgil.me" to visit, but with validate alternate link not visited.


<id="5" href="javascript:void(0)" data-wait=5000 data-url_head="https://www.wilgil.me" data-url_tail="https://wgilreath.github.io/WillHome.html" onclick="xref(id)">...</a>

Valid link "wilgil.me" exists and valid alternate link.
